Как создать репозиторий
1. Создаём каталог на компе "Name"
2. Даём команду git init
3. В каталоге "Name" заводим рабочий файл "File" на ветке Master
4. Зайти на Github - там создаём каталог "Name"
5. Копируем его адрес в HTTPS-формате
6. На компе, находясь в новом репозитории, даем команду git remote add origin "адрес с github"
6а. Чтобы привязать к другому каталогу команда: git remote rm origin "адрес"
7. git push --set-upstream origin master. После этого на github появится ветка master с файлом "File"


Порядок работы над задачей
1. Проверка обновлений git pull - сделать на каждой ветке отдельно
2. Проверка состояния git status
3. Создаём новую ветку от мастер 
3a. git checkout master
3b. git checkout -b <новая ветка>
4. Работа в новой ветке
5. Сохранение 
5а. git add .
5b. git commit -a -b "комментарий"
6. Оправка на github: git push --set-upstream origin <имя ветки>
6. если ветку уже отправляли, то: git push
